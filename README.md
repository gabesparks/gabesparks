### 👨‍💼 Who am I?

I'm a Senior Talent Partner working at the intersection of **recruiting and automation**. I spend my time finding great people, building the systems that make hiring faster and smarter, and figuring out how AI can do the repetitive stuff so I can focus on the human side of the job.

I'm not a developer — I'm a recruiter who got tired of doing things manually and decided to do something about it. Most of what you'll find here is practical, real-world tooling built to solve actual recruiting problems.

---

### 🛠️ What I'm building

| Project | Description | Stack |
|---|---|---|
| 🔍 **GitHub & NPM Sourcing Toolkit** | Four CLI tools for finding top contributors, evaluating profiles, and running a full sourcing pipeline from a single command | Node.js · GitHub API · NPM Registry |
| 📖 **GitHub Sourcing Playbook** | A structured methodology for technical sourcing on GitHub, built around the toolkit | Markdown · Documentation |
| 🐦 **Twitter/X Sourcing Playbook** | A sourcing playbook for finding crypto-native and technical developers on Twitter/X | Markdown · Documentation |
| 🤖 **TalentOps Automation** | Recruiting automation scripts for Greenhouse and Slack | Google Apps Script · Greenhouse API · Slack |
| 📋 **Weekly Roles Update** | Auto-posts open roles to Slack every Friday | JavaScript · Webhooks |
